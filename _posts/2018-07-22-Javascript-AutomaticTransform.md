: 프로그래밍 언어가 내부적으로 자동으로 자료형 변환
: Number(), String(), Boolean()등의 자료형 변환 함수를 사용하지 않고
: 연산자와 함께 사용 했을때 자료형이 변환되는 것

### 1. 문자로 자동 변환되는 경우

- 덧셈(+) 연산자를 사용하는 경우

### 2. 숫자로 자동 변환 되는 경우

- 빼기(-), 나누기(/), 곱하기(\*) 연산자를 사용하는 경우

### 3. Boolean 으로 변환되는 경우

- 논리부정연산자(!)를 사용하는 경우
- 조건문(if/switch/for/while)을 사용하는 경우
